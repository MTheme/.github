## 💡 About Us

**MTheme** is dedicated to breaking the boundaries of traditional Android home screens and exploring UI designs that are both aesthetically pleasing and highly functional. 

We primarily focus on the **Kustom (KLWP/KWGT)** dynamic wallpaper ecosystem, complemented by our in-house native Android dashboard applications and robust cloud services. Our goal is to provide users with an all-in-one, highly customizable home screen personalization experience.

> *"Redefining your home screen, one widget at a time."*

---

## 🚀 Featured Projects

### 🎨 [SILK UI](https://github.com/MTheme)
**Next-generation KLWP Preset & Dashboard**
SILK UI is our flagship home screen theme package. It is not just a KLWP preset, but comes with a standalone Android dashboard app built entirely from scratch using Jetpack Compose.
- ✨ **Rich Components**: Immersive music covers, dynamic weather curves (AQI), schedule calendars, fully automated Bilibili anime tracking, and RSS news aggregation.
- ⚡ **Seamless Interaction**: Manage image resources, change wallpapers, and configure API keys directly through the native Android app, enjoying a smooth theme-switching experience.

### 🏔️ [Edge UI](https://github.com/MTheme)
**Minimalist & Aesthetic**
A KLWP preset library focused on minimalist aesthetics. It emphasizes edge visual design, natural transition animations, and lightweight system resource consumption.

### ☁️ [MTheme Services & Bots](https://github.com/MTheme)
**The Backbone of MTheme**
The backend infrastructure that supports massive dynamic data. It includes API interfaces built with Node.js, customized RSSHub subscription services, data scraping tools, and a Discord bot engine for community management.

## 🛠️ Tech Stack & Architecture

We utilize the following modern tech stack to build our UIs and underlying systems:

- **Client Side**: `Kotlin`, `Jetpack Compose`, `Coroutines`, `Retrofit`, `C++ (JNI Security)`
- **Server Side**: `Node.js`, `Docker`, `Docker Compose`, `Redis`, `MySQL`
- **Design & Ecosystem**: `Kustom (KLWP)`, `Material Design 3`
